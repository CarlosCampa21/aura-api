# Egresados — DASC (Titulación y trámites)

## Resumen
Guía para egresados del Departamento Académico de Sistemas Computacionales (DASC) sobre titulación, modalidades con y sin examen, documentación, lineamientos de fotografías, formatos editoriales y trámites relacionados, incluyendo opciones de posgrado y reposición de documentos.

## Titulación
Para el DASC, es fundamental que sus estudiantes culminen sus estudios de licenciatura, ya que la obtención del título profesional representa la culminación de un largo camino de esfuerzo, estudio y dedicación. Contar con el título profesional abre puertas para obtener mejores puestos en organizaciones públicas o privadas, y es un requisito para ingresar a estudios de posgrado.

## Información sobre la titulación
Los requisitos para obtener el título profesional son:

- Guía de requisitos para tramitar el título profesional: Proporciona una lista detallada de los documentos y pasos necesarios para iniciar el proceso de titulación.
- Guía de requisitos especiales para tramitar el título profesional por examen profesional: Detalla los requisitos específicos para quienes optan por presentar un examen profesional como modalidad de titulación.
- Guía de requisitos especiales para tramitar el título profesional por exención de examen profesional: Indica los criterios y documentación necesaria para aquellos que buscan exentar el examen profesional.

## Documentación
Se proporcionan diversos formatos necesarios para el proceso de titulación:

- Solicitud de registro de anteproyecto: Formato para registrar el anteproyecto de tesis o trabajo profesional.
- Solicitud de aplicación de examen global teórico-práctico: Para quienes eligen esta modalidad de titulación.
- Solicitud de titulación primer autor de una publicación: Para egresados que han sido primeros autores en publicaciones científicas o de divulgación.
- Solicitud de examen profesional: Formato estándar para solicitar la presentación del examen profesional.
- Solicitud de exención de examen profesional: Para aquellos que cumplen con los criterios para exentar el examen.

## Posgrado
Para quienes continúan con estudios de posgrado, se ofrecen:

- Guía de requisitos para tramitar examen de grado (maestría o doctorado): Detalla los pasos y documentación necesaria para obtener el grado de posgrado.
- Solicitud de registro de anteproyecto de posgrado: Formato para registrar el anteproyecto correspondiente.
- Solicitud de examen de grado: Para solicitar la presentación del examen de grado.

## Instrucciones para fotografías
Se especifican las características requeridas para las fotografías que acompañan los documentos de titulación:

- Características de las fotografías tamaño infantil: Indicaciones sobre dimensiones y especificaciones técnicas.
- Características de las fotografías para grado de maestría y doctorado: Detalles específicos para niveles de posgrado.

## Formato editorial
Se proporciona un formato editorial estándar para la presentación de tesis y trabajos profesionales.

## Robo, extravío o deterioro de documentos
En caso de pérdida o daño de documentos oficiales, se ofrecen:

- Características de las fotografías para certificaciones: Especificaciones para las fotografías requeridas en procesos de reposición.
- Guía de requisitos para tramitar certificación de licenciatura y posgrado: Detalla los pasos para obtener duplicados o certificaciones.

## Modalidades de titulación
El DASC ofrece diversas modalidades de titulación, tanto con examen profesional como sin él:

### Con examen profesional
- Tesis
- Tesina
- Memoria del Servicio Social
- Memoria Técnica de Trabajo Profesional
- Curso Especial de Titulación
- Memoria de Participación en un Proyecto de Investigación
- Memoria o Prácticas Profesionales
- Producción de Materiales Educativos
- Examen Global Teórico-Práctico

### Sin examen profesional
- Créditos de maestría
- Estudios de especialización
- Promedio de 90
- EGEL-CENEVAL
- Examen Global Teórico
- Ejercicio o Práctica Profesional
- Primer autor en una publicación (revista científica o de divulgación)
- Certificación en una Norma Oficial Mexicana
- Certificaciones de Competencias Laborales

## Título sin examen profesional
Se podrá obtener el título sin presentar examen profesional cuando el alumno:

- Haya cubierto la totalidad del plan de estudios de una maestría, o
- Haya obtenido el grado de una especialidad, o
- Haya obtenido 90 de promedio global, termine sus estudios en el tiempo establecido y no haya presentado ningún extraordinario, o
- Haya obtenido un promedio global superior a 90, termine los estudios en el tiempo establecido en el plan de estudios y no haya presentado más de dos exámenes extraordinarios. En esta opción, la solicitud deberá ser aprobada por el Consejo Académico del Área, o
- Haya presentado y aprobado el examen general de egreso de licenciatura EGEL aplicado por el CENEVAL y de conformidad con el puntaje.
