# Documentos para descarga

Qué va aquí:
- Archivos originales que el usuario final puede descargar: PDFs, DOCX, PPTX, imágenes empaquetadas, etc.

Ejemplos:
- `calendario-2025.pdf`
- `reglamento-pp.pdf`
- `mapa-campus.pdf`
- `procedimiento-titulacion.docx`

Notas:
- Mantén nombres claros y en kebab-case.
- Versiona con sufijos cuando aplique: `calendario-2025-v2.pdf`.
- No coloques aquí material interno o borradores; usa solo entregables finales.
