# Recursos visuales (opcional pero recomendado)

Qué va aquí:
- Imágenes y recursos visuales que AURA podría mostrar al usuario.

Ejemplos:
- `salones/S204.jpg`
- `salones/LAB-A.jpg`
- `logos/UABCS.png`
- `imagenes-carreras/IDS.png`

Notas:
- Usa carpetas por categoría (`salones/`, `logos/`, etc.).
- Nombres descriptivos y en kebab-case cuando aplique.
- Optimiza imágenes para la web (peso y tamaño razonables).
